How to create your own google chrome extension

If yor aren a Google Chrome user, you've probably used some extensions in the browser.

What is a chrome Extension ?

A chrome exteension is a program that is installed in the chrome brower that enhanced the functionality of the browser. you can build one easily using web technologies like html css and javascript.

Creating a chrome extension is similar to a web application, but it requires a mainifest.json. file.
